## ==1) auth==

### ==1.1 `POST /api/v1/auth/code`==
**Кто:** Гость  
отправить одноразовый код на email для входа/регистрации

**Body:**
```json
{
  "email": "user@example.com"
}
```

**Ответ:**
- `204 No Content`

---

### ==1.2 `POST /api/v1/auth/verify`==
**Кто:** Гость  
подтвердить код
Если пользователя нет,  он создаётся
(напоминание: при создании автоматически создаются 3 системные папки)

**Body:**
```json
{
  "email": "user@example.com",
  "code": "123456"
}
```

**Ответ:**
```json
{  
"access_token": "jwt_token",  
"refresh_token": "refresh_token",  
"token_type": "bearer",  
"user": {  
	"id": "uuid",  
	"email": "user@example.com",  
	"name": "User",  
	"role_id": 1  
  }  
}
```

---

### ==1.3 `POST /api/v1/auth/refresh`==
**Кто:** Пользователь  / Админ
получить новый access-токен по refresh-токену

**Body:**
```json
{
  "refresh_token": "refresh_token"
}
```

**Ответ:**
```json
{
  "access_token": "new_jwt_token"
}
```

## 2) places

### ==2.1 `GET /api/v1/places==
**Кто:** Гость / Пользователь  / Админ
получить список мест с поиском и фильтрацией (опубликованные)

**Query параметры:**
- `q` — поиск по названию
- `city` — фильтр по городу
- `tag` — фильтр по тегам
- `page`, `limit` — пагинация
- `sort` — сортировка

**Ответ:**
```json
{
  "items": [
    {
      "id": "uuid",
      "name": "Название",
      "city": {"id": "uuid",
		       "city": "Москва"},
      "cover_photo": "https://...",
      "tags": [
			    {
			      "id": "uuid",
			      "name": "кофейня"
			    }
		    ]
    }
  ],
  "page": 1,
  "limit": 10,
  "total": 120
}
```

----
